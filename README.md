# oop-projects
This repository contains the projects developed throughout the Object-Oriented Programming course at the Federal University of Rio de Janeiro (UFRJ). The primary goal is to develop a deep and practical understanding of the object-oriented paradigm.

To strengthen the learning process, the projects initially followed a low-level workflow: all code was written using Neovim as a text editor and compiled directly from the terminal. This approach helped reinforce the fundamentals of the language and the compilation process before introducing more advanced development tools.

In addition, this repository also serves as an opportunity to practice working with Git, including version control, commit organization, and maintaining a structured development history.

After building a solid foundation, the workflow transitioned to an integrated development environment (IDE), specifically IntelliJ IDEA, allowing for increased productivity while maintaining a clear understanding of the underlying mechanisms.

By combining a strong conceptual foundation with modern development tools and version control practices, this repository reflects a learning process designed not only to master Java, but also to build transferable knowledge that can be applied to any programming language that supports the object-oriented paradigm.
